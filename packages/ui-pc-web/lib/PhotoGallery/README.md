# 素材库组件

参数

| 参数     | 说明             | 类型    | 可选值     | 默认值 |
| -------- | ---------------- | ------- | ---------- | ------ |
| mask     | 遮罩是否可以点击 | Boolean | true,false | true   |
| maxCount | 最多选择项       | number  | ---        | 9      |

方法

| 方法名      | 说明               | 返回值 |
| ----------- | ------------------ | ------ |
| openPhoto   | 打开素材库         | 无     |
| closePhoto  | 关闭素材库         | 无     |
| returnPhoto | 选择完图片确定触发 | 数组   |
